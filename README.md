# WID2002-Clothes-Matching-System
## Installation
- Create venv with the name venv
```Shell
py -m venv venv
```
- Activate venv
```Shell
venv\Scripts\activate
```
- Install required libraries
```Shell
py -m pip install -r requirements.txt
```
- Update requirements.txt
```Shell
pip freeze > requirements.txt
```
